# Corpus: BJMNS (`bjmns.tumblr.com,2013`)

This is the original corpus of my works developed prior to my graduating college.
It used a somewhat minimal Project.Volume.Version.Draft(.Verse) system which is preserved here as best as possible.

\[Note “Verse” was referred to as “Part” during the terminology of the time.]

Serialization was generally done as follows:

    000 ProjectID
    Volume I – “Some Volume”
    Version 2 – “Some Version”
    Draft 03 – “Some Draft”
    Verse i – “Some Verse”

\[Again, “Verse” actually would have been “Part” at the time.]

For full paths, a simple decimal system was used:

    0.1.2.3
    0.1.2.3.1

This reflects (and explains) the opaque directory structure used here.
